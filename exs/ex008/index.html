Principais formatações

Negrito ou destaque

Uma frase utilizando um termo em negrito. Para que isto funcione, utilizo o termo b (bold).

Porém, o uso da tag b não é semântico, sendo substítuido pela tag strong.

Itálico

Para colocar algum termo em itálico basta usar o termo i. Não semântico.

Com a semântica, colocar a tag en, para colocar em itálico.

Texto Marcado

Também é possível realçar o fundo de um termo usando a tag mark.

Abreviação

Para comprir algum texto e abreviar uma palavra, é legal usar a tag abbr. Por exemplo no HTML
